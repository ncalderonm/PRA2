Descripcin del dataset:

Nuestros datos se dividen en 3 grupos:

  • train.csv
  • test.csv
  • gender_submission.csv

El set de datos train, nuestra base de datos de entrenamiento, se utilizar para construir nuestro modelo de prediccin. Este set de datos contiene informacin relativa a 891 pasajeros que estuvieron en el Titanic, as como informacin sobre si sobrevivieron o no.

En este trabajo, buscaremos analizar las diferentes condiciones y relaciones que pudieron determinar que un pasajero sobreviviera o no a ese suceso. Por ejemplo, analizaremos si ciertos factores como el sexo de la persona, la edad o incluso su estatus social (entre otras) fueron decisivas para su supervivencia. A partir de este anlisis, se plantear un modelo capaz de predecir la supervivencia de los pasajeros en funcin de estas caractersticas.

Para probar la consistencia del modelo, se utilizar el set de datos test. Es por esto que este set de datos no contiene informacin sobre si el pasajero ha sobrevivido o no, ya que deberemos predecirlo a partir de nuestro modelo.

Finalmente, a partir del archivo gender_submission (que contiene los verdaderos datos de supervivencia del set de datos test), evaluaremos la precisin de nuestro modelo en cuanto a las predicciones obtenidas a partir del set de datos test.

Wilcoxon:

Dado que intuimos que las variables no se distribuyen como una normal, aplicaremos el test no paramtrico de Wilcoxon que nos ayudar a comparar el rango medio entre dos muestras y determinar si existen diferencias entre ellas, en nuestro caso las muestras se dividirn segn nuestra variable objetivo Survived.

Por otro lado, si nuestras variables se distribuyesen segn una normal, el t-test sera el test estadstico paramtrico que parmitira contrastar la hiptesis nula de que las medias de dos poblaciones son iguales, frente a la hiptesis alternativa de que no lo son.

Vemos con Wilcoxon:

# Semilla:
set.seed(333)

# Incluimos la variable objetivo:
titanic_train_num_norm$Survived <- titanic_train$Survived

# Aplicamos el test a nuestras variables numricas normalizadas:
for (i in colnames(titanic_train_num_norm[,c("Age", "SibSp", "Parch", "Fare")])) {
  test <- wilcox.test(titanic_train_num_norm[, i] ~ titanic_train_num_norm$Survived)
  title <- sprintf("Test de Wilcoxon para %s vs Survived", i)
  print(title)
  print(test)
  cat("\n\n")
}

Antes de tratar las conclusiones, comentamos cmo interpretar Phi y Cramer:

  • Phi: El coeficiente phi es el valor de la chi cuadrado entre el nmero de observaciones, un valor prximo a 0 indica independencia entre los factores, valores prximos a 1 implican relacin entre los factores.

  • CramerV o V de Cramer: Muy habitual para medir la relacin entre factores. Tambin 0 implica independencia y 1 una relacin perfecta entre los factores.

Para ambos test, valores superiores a 0,30 nos indican que hay una posible relacin entre las variables.

Conclusiones:

  • Los coeficientes son iguales, al ser tablas de contingencia de 2x2.

  • Las correlaciones relevantes encontrada se dan entre nuestra variable objetivo y Sex y en menor medida Pclass (al presentar un coeficiente mayor a 0.3). Esto adems se puede intuir/comprobar en la tabla de contingencia en relativo, por ejemplo para la variable Sex, se observa como el 81% de los hombres no sobreviven, mientras que el 75% de las mujeres s lo hacen.

Homocedasticidad:

Despus de realizar un test de Shapiro-Wilks sobre los datos y ver que estos no se distribuyen siguiendo una normal, se realizar un test para comprobar la homocedasticidad de los datos. El supuesto de homocedasticidad considera que la varianza es constante, es decir, no vara entre diferentes grupos. Como no se cumple la condicin de normalidad en los datos, se opta por realizar un test de Fligner-Killeen, un test no paramtrico que compara las varianzas basndose en la mediana.

# Semilla:
